## Graceful Fallbacks

The system degrades gracefully when optional components are unavailable:

```
SONA unavailable  → No-op with dummy ID (-1)
GNN unavailable   → Uniform weights (1/k each)
GraphStore error  → Vector-only scoring continues
Embedding ONNX    → Hash-based fallback (functional)
```
