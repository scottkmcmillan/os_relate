## Architecture Changes

### Previous Architecture (Next.js)
```
Frontend: Next.js 14 + React 18 + TypeScript
Backend: Python/FastAPI
Database: PostgreSQL + RuVector extension
Auth: Custom JWT implementation
```

### New Architecture (Lovable)
```
Frontend: Lovable (Vite + React + TypeScript + Shadcn UI)
Auth/Users: Supabase (Auth + PostgreSQL)
Business Logic: Python/FastAPI (unchanged)
Strategic Data: PostgreSQL + RuVector extension (vectors + hypergraph)
Real-time: Supabase Realtime
File Storage: Supabase Storage
```
